Array Of Object

1. Khái niệm về mảng

  • Mảng (Array) là một cấu trúc dữ liệu lưu trữ tập hợp các phần tử có cùng kiểu dữ liệu (int, float, char, object, …).
  • Các phần tử được lưu trữ trong bộ nhớ liên tiếp và được truy cập thông qua chỉ số (index), bắt đầu từ 0.

2. Cách khai báo và khởi tạo mảng

Khai báo mảng:

Code 

      dataType[] arrayName;

Khởi tạo mảng:

  • Khởi tạo trực tiếp với kích thước:
Code 

     int[] numbers = new int[5]; // Mảng gồm 5 phần tử, mặc định giá trị là 0.
  • Khởi tạo trực tiếp với giá trị:
Code 

     int[] numbers = {1, 2, 3, 4, 5};
  • Khởi tạo mảng bằng cách gán giá trị sau:
Code 

     int[] numbers = new int[5];
     numbers[0] = 10;
     numbers[1] = 20;

3. Các thao tác cơ bản trên mảng

In mảng

  • Tự in từng phần tử:
Code 

     for(int i = 0; i < arr.length; i++) {
       System.out.print(arr[i] + " ");
     }
  • Sử dụng Arrays.toString():
Code 

     import java.util.Arrays;  
     int[] arr = new int[5]
     System.out.println(Arrays.toString(arr)); // [0, 0, 0, 0, 0]

Sắp xếp mảng

Code 

     import java.util.Arrays;  
     int[] numbers = {1, 2, 3, 4, 5};    
     Arrays.sort(numbers);
     System.out.println(Arrays.toString(numbers)); // [1, 2, 3, 4, 5]

Tìm kiếm phần tử trong mảng

  • Tìm kiếm tuyến tính
  • Tìm kiếm nhị phân
Code 

     //Yêu cầu mảng phải được sắp xếp trước.
     int key = 4;
     int index = Arrays.binarySearch(numbers, key);
     System.out.println("Found at index: " + index);

4. Mảng động

Java không hỗ trợ mảng động trực tiếp, nhưng bạn có thể sử dụng lớp ArrayList để mô phỏng mảng động.

Code 

     import java.util.Arrays;
     ArrayList<Integer> list = newArrayList<>();      
     list.add(1);
     list.add(2);
     list.remove(0);
     System.out.println(list); // [2]

5. Ưu nhược điểm của mảng

Ưu điểm:

Dễ sử dụng và hiệu quả trong việc lưu trữ dữ liệu đồng nhất. Tốc độ truy cập nhanh.

Nhược điểm:

Kích thước cố định, không thể thay đổi trong quá trình chạy. Khó quản lý và thao tác khi kích thước lớn.
