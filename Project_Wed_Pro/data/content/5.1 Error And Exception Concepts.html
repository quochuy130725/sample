Error & Exception Concepts

1. Khái niệm về Error & Exception Concepts

  • Error: Lỗi không thể phục hồi (e.g., OutOfMemoryError).
  • Exception: Lỗi có thể xử lý trong chương trình.
  • Loại exception:
    • Checked: Cần xử lý tại compile-time (e.g., IOException).
    • Unchecked: Phát sinh tại runtime (e.g., NullPointerException).

Ví dụ về Error (Lỗi không thể phục hồi)

Code 

            public class ErrorExample {
                public static void main(String[] args) {
                    try {
                        // Cố gắng tạo ra một lỗi OutOfMemoryError
                        int[] largeArray = new int[Integer.MAX_VALUE];
                    } catch (Error e) {
                        System.out.println("Caught Error: " + e);
                    }
                }
            }

Ví dụ về Checked Exception (Lỗi cần xử lý tại compile-time)

Code 

            import java.io.*;
            
            public class CheckedExceptionExample {
                public static void main(String[] args) {
                    try {
                        // Cố gắng đọc từ một tệp không tồn tại
                        FileReader file = new FileReader("nonexistentfile.txt");
                        BufferedReader fileInput = new BufferedReader(file);
                        System.out.println(fileInput.readLine());
                        fileInput.close();
                    } catch (IOException e) {
                        System.out.println("Caught Checked Exception: " + e);
                    }
                }
            }
            
//Caught Checked Exception: java.io.FileNotFoundException: nonexistentfile.txt (No such file or directory)

Ví dụ về Unchecked Exception (Lỗi phát sinh tại runtime)

Code 

            public class UncheckedExceptionExample {
                public static void main(String[] args) {
                    try {
                        String str = null;
                        // Cố gắng gọi phương thức trên đối tượng null
                        System.out.println(str.length());
                    } catch (NullPointerException e) {
                        System.out.println("Caught Unchecked Exception: " + e);
                    }
                }
            }
Tóm tắt :
    Error là lỗi nghiêm trọng không thể phục hồi (ví dụ: OutOfMemoryError).
  • Checked Exception là lỗi mà bạn phải xử lý tại compile-time (ví dụ: IOException).
  • Unchecked Exception là lỗi mà chỉ phát sinh tại runtime (ví dụ: NullPointerException).
