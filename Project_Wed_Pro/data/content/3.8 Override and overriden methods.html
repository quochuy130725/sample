Override và Overridden Methods

1. Khái niệm về Override

  • Override là khi 1 class con kế thừa từ class cha funtions nhưng không muốn hoạt động theo cách giống class cha (supper class)

2. Cách triễn khai Override

Code 

            class Animal {
                void sound() {
                    System.out.println("Animal makes a sound"); // Phương thức sound trong lớp Animal
                }
            }
            
            class Dog extends Animal {
                @Override
                void sound() {
                    System.out.println("Dog barks"); // Phương thức sound trong lớp Dog (ghi đè từ lớp Animal)
                }
            }
            
            Animal myDog = new Dog(); // Tạo đối tượng Dog kiểu Animal
            myDog.sound(); // Output: Dog barks
