Giới thiệu về lớp (class) trong Java

1. Ví dụ về lớp (class)

Code 
            class Student {
                String name;
                int age;
            
                public void display() {
                    System.out.println("Name: " + name);
                    System.out.println("Age: " + age);
                }
            }
            
            public class Entry {
                public static void main(String[] args) {
                    Student student1 = new Student();
                    student1.name = "Linh";
                    student1.age = 24;
                    student1.display();
                }
            }

Ví dụ này giúp bạn hiểu một số thành phần cơ bản của lớp (class):

Lớp (class): Là nơi định nghĩa thông tin về các đối tượng. Bạn có thể hiểu lớp là một kiểu dữ liệu mà biến của kiểu này được coi như một đối tượng.

Ví dụ: Học sinh, giáo viên, xe ô tô, ...

Đối tượng (object): Là một thể hiện của lớp. Ví dụ, student1 là một đối tượng đại diện cho thông tin của một học sinh.

Thuộc tính (attribute): Là các đặc điểm của đối tượng. Ví dụ, name và age là hai thuộc tính của đối tượng học sinh.

Phương thức (method): Là hành vi của đối tượng. Ví dụ, display() là một phương thức dùng để giới thiệu bản thân của học sinh.

            class Student {
                String name;

                }

4. Tạo đối tượng từ lớp

Để tạo đối tượng từ lớp, bạn dùng từ khóa new. Ví dụ:

Code 
            Student s1 = new Student();
            s1.name = "Long";
            s1.display();
