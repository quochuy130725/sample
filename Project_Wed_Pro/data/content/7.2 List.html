7.2 List

Sơ lược về List

  • Duy trì thứ tự phần tử khi thêm vào.
  • Cho phép chứa các phần tử trùng lặp.
  • Các lớp triển khai phổ biến:
    • ArrayList: Mảng động có hiệu suất tốt.
    • Vector: Giống ArrayList nhưng hỗ trợ đồng bộ (synchronized).
    • LinkedList: Triển khai danh sách liên kết, thích hợp khi cần thêm/xóa thường xuyên.

Các phương thức cơ bản

Thêm phần tử: add(E element) hoặc add(int index, E element)
Code 

   import java.util.ArrayList;
   ArrayList<String> list = new ArrayList<>(); 
   list.add("Apple");  
   list.add("Banana"); 
   list.add(2, "Cherry"); 
   System.out.println("Danh sách: " + list);  //Danh sách: [Apple, Banana, Cherry]
Xóa phần tử: remove(int index) hoặc remove(Object o).
Code 

   import java.util.ArrayList;
   ArrayList<String> list = new ArrayList<>(); 
   list.add("Apple");  
   list.add("Banana"); 
   list.add("Cherry");
   list.remove("0");  
   //hoặc
   list.remove("Apple");  
   System.out.println("Danh sách: " + list);  //Danh sách: [Banana, Cherry]
Tìm kiếm phần tử: contains(Object o) để kiểm tra sự tồn tại sẽ trả về giá trị boolean.
Code 

   import java.util.ArrayList;
   ArrayList<String> list = new ArrayList<>(); 
   list.add("Apple");  
   list.add("Banana"); 
   list.add("Cherry");
   System.out.println(list.contains(“Cherry”)); //true
Sắp xếp các phần tử: sử dụng Collections.sort(List<T> ) hoặc cung cấp Comparator
Code 

   import java.util.ArrayList; 
   import java.util.Collections; 
   ArrayList<String> list = new ArrayList<>(); 
   list.add("Apple");  
   list.add("Cherry");
   list.add("Banana"); 
   System.out.println("Danh sách trước khi sắp xếp: " + list); //Danh sách trước khi sắp xếp: [Apple, Cherry, Banana]                 
   // Sắp xếp danh sách
   Collections.sort(list);
   System.out.println("Danh sách sau khi sắp xếp: " + list); //Danh sách sau khi sắp xếp: [Apple, Banana, Cherry]
