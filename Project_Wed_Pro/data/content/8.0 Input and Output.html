Input and Output

Stream trong Java

Stream trong Java là một cơ chế để đọc, ghi, và xử lý dữ liệu từ các nguồn khác nhau (file, console, network, v.v.).
Nó là nền tảng cho I/O (Input/Output) trong Java.

1. Phân loại Stream:

  • Java cung cấp hai loại stream chính:
    1. Byte Stream:
      • Xử lý dữ liệu ở dạng byte (8-bit).
      • Dùng cho dữ liệu nhị phân như hình ảnh, âm thanh, video.
      • Các class tiêu biểu:
        • InputStream (đọc dữ liệu byte).
        • OutputStream (ghi dữ liệu byte)
    2. Character Stream:
      • Xử lý dữ liệu ở dạng ký tự (16-bit Unicode).
      • Dùng cho dữ liệu văn bản (text).
      • Các class tiêu biểu:
        • Reader (đọc dữ liệu ký tự).
        • Writer (ghi dữ liệu ký tự).
