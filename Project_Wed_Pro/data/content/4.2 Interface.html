Interface

1. Khái niệm về Interface

  • Là một bảng hợp đồng giữa các class.
  • Nó bắt buộc các class phải triễn khai Override toàn bộ các phương thức trong nó

2. Cách triễn khai Interface

Code 

            interface Animal {
                void eat(); // Phương thức ăn
            }
            
            interface Bird {
                void fly(); // Phương thức bay
            }
            
            class Eagle implements Animal, Bird {
                public void eat() {
                    System.out.println("Eagle eats"); // Triển khai phương thức eat
                }
            
                public void fly() {
                    System.out.println("Eagle flies"); // Triển khai phương thức fly
                }
            }
            
            Eagle eagle = new Eagle(); // Tạo đối tượng Eagle
            eagle.eat(); // Output: Eagle eats
            eagle.fly(); // Output: Eagle flies
