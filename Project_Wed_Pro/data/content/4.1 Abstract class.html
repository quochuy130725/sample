Abstract Class

1. Khái niệm về Abstract Class

  • Là những gì thuộc về sơ khai, của chung nhất VD: con người , động vật , thực vật
  • Tính trừu tượng trong lập trình là quá trình ẩn đi các chi tiết triển khai không cần thiết, chỉ để lại các đặc điểm quan trọng, giúp giảm độ phức tạp của hệ thống.

2. Cách triễn khai Abstract Class

Code 

            abstract class Shape {
                abstract void draw(); // Phương thức trừu tượng
            }
            
            class Circle extends Shape {
                void draw() {
                    System.out.println("Drawing a circle"); // Ghi đè phương thức draw
                }
            }
            
            Shape shape = new Circle(); // Tạo đối tượng Circle kiểu Shape
            shape.draw(); // Output: Drawing a circle
