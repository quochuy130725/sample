7.4 Map

Sơ lược về Map

  • Chứa các cặp key-value.
    VD: trong map lưu 1 đối tượng là (0, 2) thì 0 là key và 2 là value.
  • Mỗi key là duy nhất, value có thể trùng lặp.
  • Các lớp triển khai phổ biến:
    • HashMap: Lưu trữ key-value trong bảng băm, thứ tự không được đảm bảo.
    • TreeMap: Sắp xếp key theo thứ tự tự nhiên.

Các phương thức cơ bản

Thêm phần tử: put(K key, V value) Set sẽ không thêm nếu phần tử đã tồn tại.
Code 

   import java.util.HashMap;
   HashMap<Integer, String> map  = new HashMap<>(); 
   map.put(1, "One");  
   map.put(2, "Two");
   System.out.println("Ánh xạ: " + map);  //Ánh xạ: {1=One, 2=Two}
Xóa phần tử: remove(Object key).
Code 

   import java.util.HashMap;
   HashMap<Integer, String> map  = new HashMap<>(); 
   map.put(1, "One");  
   map.put(2, "Two");
   map.remove(1);
   System.out.println("Ánh xạ: " + map);  //Ánh xạ: {2=Two}
